Welcome!

Esta aplicación se enfoca en implementar y evaluar diferentes algoritmos de aprendizaje automático y redes neuronales, para el análisis de polaridad (positivo/negativo) en reseñas cinematográficas. Utilizando como dataset IMDB y SST-2, se busca desarrollar un pipeline completo que aborde todas las etapas del procesamiento de texto.

Arquitectura basada en Flask, Gitlab y Python para el procesamiento de datos.
